# accreditation_disk_bot_v2

Вторая обновленная итерация бота для загрузки файлов на Яндекс.Диск хранилище Первого МГМУ им. Сеченова для сотрудников.

## Функциональность

- Выбор папки
- Пагинация кнопок папок
- Загрузка файла в выбранную папку
- Индикация окончания загрузки

## Зависимости

- Docker

## Деплой

```bash
sudo chmod +x deploy.sh
./deploy.sh
```
